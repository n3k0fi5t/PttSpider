# PttSpider
## Introduction
A Ptt Spider in charge of generating Data for Django blog website tutorial
- PttArticleListSpider
fetch article urls of specific ptt board
    
- PttArticleSpider
fetch:
    - article title
    - author
    - post time
    - image list
    - push list
